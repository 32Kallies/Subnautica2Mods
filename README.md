# Kallie23's Subnautica 2 Mods

All mods in this repository are Lua mods developed for Subnautica 2, built to run on UE4SS.

Requires [UE4SS for Subnautica 2](https://github.com/Subnautica2Modding/Subnautica2-UE4SS/releases).

## Installation

1. If you have not done so yet, please first download and install UE4SS for Subnautica 2.
2. Download the desired mods. You can get these on my [Nexus Mods account](https://www.nexusmods.com/profile/Kallie23/mods) (where I tend to upload stable downloads), or by manually downloading them from the repository.
3. Place the mod folder(s) you downloaded into the `Subnautica2\Subnautica2\Binaries\Win64\ue4ss\Mods` folder.

# Mods

## Expanded Inventory Mod

Unlocks all possible inventory slots at the start of the game. This change is not reversible. While the developers intended for players to unlock these naturally over time,
this mod particularly helps if you missed a location and don't want to go looking for it.

## SN2 Commands Enabler Mod

Activates the in-game command console. You can open this with the Enter key.

> [!WARNING]
> It is very easy to accidentally misuse the command console. I do not suggest using this in multiplayer unless everyone in the game is willing to take the risks. 

<img width="741" height="265" alt="image" src="https://github.com/user-attachments/assets/5ea2e5d7-4af8-4789-8921-32daeabe6225" />

## SN2 Developer Maps Enabler
Enables the 'Developer Game' button on the main menu that lets you load into various developer testing maps. This can be useful for quickly testing mods and exploring some unused/hidden content.
